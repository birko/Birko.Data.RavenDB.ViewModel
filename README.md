# Birko.Data.RavenDB.ViewModel

RavenDB ViewModel repository implementations for the Birko Framework.

## Features

- ViewModel repositories for RavenDB
- Multiple constructor variants for connection flexibility

## Installation

```bash
dotnet add package Birko.Data.RavenDB.ViewModel
```

## Dependencies

- Birko.Data.Core (AbstractModel, ViewModels)
- Birko.Data.Stores (store interfaces)
- Birko.Data.Repositories (repository interfaces)
- Birko.Data.ViewModel
- Birko.Data.RavenDB
